Diccionario de Apuestas Inteligentes

Comprende la terminología detrás del modelo matemático y úsalo a tu favor.

1. Value Betting (EV+)

El Valor Esperado (EV) ocurre cuando el cálculo de probabilidad del modelo IA sobrepasa el porcentaje implícito marcado por la Casa de Apuestas. Si la casa paga @2.00 (50% de prob), pero la IA dicta que el Local gana el 60% de las veces, tienes un valor a largo plazo y una ventaja matemática ineludible.

2. Criterio de Kelly Multiplicador

Incluso la mejor apuesta no debe llevarte a la bancarrota. La fórmula de Kelly especifica qué % de tu Bankroll destinar a una cuota EV+ acorde a su riesgo. El sistema la multiplica artificialmente por 0.1 (Kelly Fraccional Conservador) bajando tu varianza y maximizando la supervivencia.

3. Expected Goals (xG)

A diferencia de los goles reales que dependen mucho de la suerte o el rendimiento de un portero estrella, los xG miden la "calidad real" y asiduidad de los tiros generados. Nuestro motor penaliza o bonifica si percibe que un equipo anota por suerte o si genera sin premio.

Manual Técnico

Arquitectura, flujos de datos y configuración del sistema PredictScore.

Arquitectura General

PredictScore sigue una arquitectura ETL (Extracción-Transformación-Carga) automatizada con las siguientes capas:

Ingesta de Datos

Football-Data.co.uk (historial 5 años) + TheOddsAPI (cuotas en vivo)

Motor IA

HistGradientBoosting + Poisson Multivariable (scikit-learn)

Backend

Flask + SQLite + Pandas para APIs REST del dashboard

Notificaciones

Telegram Bot API — 3 envíos diarios automatizados vía GitHub Actions

Flujo de Ejecución Diaria

El pipeline se ejecuta automáticamente 3 veces al día mediante GitHub Actions (bot_diario.yml):

  1. cartelera_automatica.py — Escanea la API de cuotas para partidos del día siguiente
  2. predicciones_reales.py — Entrena el modelo y genera predicciones con EV, Poisson y Kelly
  3. alerta_telegram.py — Envía el reporte formateado al canal de Telegram

Archivos Principales

ingesta_datos.pyETL: descarga histórico de 5 años de las Top 5 ligas + Champions
ingenieria_caracteristicas.pyFeature Engineering: genera xG, rachas, diferenciales
modelos_poo.pyModelos ML: GradientBoosting, RandomForest, Poisson
cartelera_automatica.pyGenera la cartelera diaria cruzando cuotas con stats
predicciones_hoy.pyPredice resultados usando el motor entrenado
evaluacion_resultados.pyConcilia predicciones vs resultados reales
simulador_kelly.pySimulador de gestión de riesgo con criterio Kelly
app.pyServidor Flask que sirve el dashboard y APIs

Variables de Entorno Requeridas

ODDS_API_KEY — Clave de TheOddsAPI para cuotas en tiempo real
TELEGRAM_BOT_TOKEN — Token del bot de Telegram
TELEGRAM_CHAT_ID — ID del chat/canal de destino

Base de Datos

Se utiliza SQLite (apuestas.db) para almacenamiento estructurado. Las tablas principales son: partidos_historicos y champions_league. El cache JSON (stats_actuales.json) se genera para acceso rápido sin consultas SQL.

Requisitos del Sistema

Python 3.10+, Flask, Pandas, scikit-learn, requests, BeautifulSoup4, scipy. Instalar con pip install -r requirements.txt.

Preguntas Frecuentes

¿Cuántas veces al día recibo alertas por Telegram?

El sistema envía 3 alertas diarias automáticas: una matutina (8:00 AM UTC), una de tarde (2:00 PM UTC) y una nocturna (8:00 PM UTC). Cada envío incluye los picks de mayor valor esperado.

¿Por qué aparecen solo partidos de La Liga?

El sistema cubre las 5 grandes ligas europeas + Champions League. Si solo ves La Liga, puede deberse a que la API de cuotas no tiene partidos programados de otras ligas ese día, o a que la clave de API necesita renovarse.

¿Qué significa EV+ (Valor Esperado positivo)?

EV+ indica que la probabilidad real calculada por la IA es mayor que la implícita de la casa de apuestas. Un EV > 1.10 es una señal de inversión rentable a largo plazo.

¿Cómo reporto un error en las predicciones?

Si detectas un falso positivo o un equipo que no refresca sus estadísticas, puedes abrir un reclamo usando el botón de abajo. También puedes contactarnos directamente por el canal de Telegram.
